1. Quais as diferenças entre os barramentos de dados e de endereços?
	o barramento de dados é usado para transmitir informaçao entra as unidades, sendo que o barramento de dados é bi-direcional. O barramento de endereços é ultizado para 
	informar o destino ou a origem dos dados, desta forma funcionamento em conjunto com o barramento de dados, logo a memoria de endereços indica o destino ou origem dos dados 		os quais serão transmitidos pelos barramentos de dados.

2. Quais são as diferenças entre as memórias RAM e ROM?
	A memoria ROM é uma memoria nao volatil, ou seja quando a energia é retirada a informação é mantida.  A memoria RAM é uma memoria volatil, ou seja é uma memoria ultilizada
	quando o computador ja esta em funcionamento. A memoria RAM tambem é conhecida como randam access memory, ou seja  esta memoria pode ser acesada pelo sistema operacional
	em qualquer momento.
